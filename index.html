Jekxyl

A jekyll like static blog generator with Hoa. Use power of XYL !

Getting started

  • install Hoa
  • git clone or download Jekxyl
  • modify layout in 
    ./Application/In/Layouts
  • write your blog posts in 
    ./Application/In/Posts
  • run 
    ./Bin/jekxyl
  • that's it, your blog is written in 
    ./Application/Out/
    folder

Metas

Jekxyl uses metas in post file to configure output. Metas are removed from HTML output.

layout

Meta layout is used to set the layout used for the current blog post. value should match the name of the layout file in Application/In/Layouts.

<?xyl-meta name="layout" value="main"?>

title

Meta title is used to set the title for the current blog post. It is used for the <title> tag of the page.

<?xyl-meta name="title" value="First post title"?>
